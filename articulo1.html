Historia de HTML

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos.




¿Qué incluye la estructura HTML?
La estructura HTML de una página web se compone de las etiquetas 'html', que indica el comienzo de la página; 'head', que contiene información más descriptiva sobre el sitio; y 'body', que conforma todos los elementos visibles de la página.

¿Qué es el diseño en HTML?
El diseño web es un área enfocada en el desarrollo de interfaces digitales, como el diseño de sitios y aplicaciones para web. Para ello, los diseñadores web crean las páginas utilizando lenguajes de marcado.

¿Cuáles son los tipos de HTML?
Los estándares oficiales HTML son el HTML 2.0, el HTML 3.2, el HTML 4.0, el HTML 4.01 y el HTML 5. El HTLM 5 es la última especificación oficial y se espera que continúe evolucionando a lo largo de los próximos años.
